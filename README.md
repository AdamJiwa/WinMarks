# WinMarks

## Overview

**WinMarks** allows you to set hotkeys for specific windows on Mirosoft Windows
similar to the marks system in VIM

## Requirements

Visual c++ runtime XX

## Building

```powershell
mkdir build
cd build
cmake ../
cmake --build .
```
